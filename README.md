

# Flood Prediction using Machine Learning 🌧️📊

## Overview
This project focuses on predicting flood occurrences using Machine Learning techniques on a real-world flood dataset. Floods are among the most destructive natural disasters, causing severe damage to life, property, and infrastructure. This system helps in early flood prediction by analyzing environmental and historical factors.

The project applies multiple supervised learning algorithms, compares their performance, and selects the most accurate model for reliable predictions.

## Features
✅ Data Cleaning & Preprocessing  
✅ Exploratory Data Analysis (EDA)  
✅ Multiple ML Model Training  
✅ Hyperparameter Tuning using RandomizedSearchCV  
✅ Model Performance Comparison  
✅ Accuracy, Precision, Recall & F1-Score Evaluation  
✅ Visualization of Results
